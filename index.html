JS Example Of How To Add HTML Elements to a Page

The Setup

For this totorial, we are going to start with a blank html file with a <ul> tag in the body.

                
                    <!doctype html>
                    <html>
                        <body>
                            <ul>
                            </ul>
                        </body>
                    </html>

After your document is created, go ahead and assign an id to your <ul>... I will be giving it an id of "shopping".

Creating The Script Tag/Array

Just underneath your closing <ul> tag, add a opening and closing <script>. The first thing we are going to do in our script is create an array. An array is like a database or a bank of values under one name, or in other words we can set a variable but allow it to have multiple values. In this example we are going to have our variable be named fruit, and the value of fruit is going to be: Apples, Oranges, Bananas, Strawberries, Cherries, Blueberries, and Watermelons.

In your text editor inbetween your opening and closing <script> you are going to add:

                
                    <script>
                        var fruit = ["Apples", "Oranges", "Bananas", "Strawberries", "Cherries", "Blueberries", "Watermelons"];
                    </script>

One thing to note about arrays is that, the counter of an array always starts at 0. So "Apples" would be an array index of 0, "Oranges" would be an array index of 1, and so on and so forth. This is important to remember for later.

Creating The Function

The next thing we have to do is create and name our function. In this case I am going to call it createList since our goal is to add <li> to our <ul>.

Just under the array you just made, input:

                
                    function createList() {
                    
                    }

Creating Our For Loop

To make things easy on us so that we aren't manual inputting all of these <li> one by one, and to be able to reuse this code in other scenarios (you would have to change some variable names) we are going to use a for loop.

Inside the curly braces of your function you are going to add:

                
                    for(i = 0; i < fruit.length; i++) {

                    }

Let's break down what this loop is telling us. Starting with the i = 0;, the variable i is common practice to stand for iteration and is saying that we want to set our initial counter to 0. This is important because when we pull in our fruit array we want it to start with "Apples", like we dicsussed up above. The i < fruit.length; is our condition. While this condition is true it is going to continue to extecute the code that we put in next until it is not true or false. In this case as long as our counter(i) is smaller than the length of our fruit array, this will continue to execute. Finally, i++. What this is telling the loop to do is to add 1 to i after the last line of code has been run within the curly braces({}) of the for loop.

Setting Variables For Our Loop

The first thing we are going to do, is within our for loop is add two variables.

Inside the curly braces of your for loop you are going to add:

                
                    for(i = 0; i < fruit.length; i++) {
                        var list = document.getElementById("shopping");
                        var item = document.createElement("li");
                    }

Our first variable is list and set it to be the id we set earlier of shopping which is attached to our <ul>. Our second variable is where the magic starts to begin. The variable is item and we are creating the <li> that will be displayed; however, although we created it, it has yet to be attached to anything and it has no content, so it will not appear on screen.

Adding Content and Attaching them To The List

With our <li> created, we now have to fill them with content and attach them to our list.

Just after the two variables that you just created, still within the curly braces({}) add:

                
                    item.innerHTML = fruit[i];
                    list.appendChild(item);

Starting with the first line, we are setting the content inside the <li>, which in our case is our fruit variable. The brackets([]) beside the fruit variable is telling it to grab the value equal to i. So, because i is starting at 0 the first <li> that will displayed is "Apples". The second time this loop gets run, the <li> will be "Oranges". With the content inputted into the <li>, we still won't see it displayed on the screen. This is because although created and it has content we have not placed it anywhere in the HTML document. This is where the last line comes in. What it is saying, is that I want to add this <li> or item (since that is the variable name that we gave it) to the <ul> with the id of "shopping" or list (also the variable name that we created).

Finishing Touches

We are almost done, but you have probably noticed that nothing is still showing up on the page. This is because right now we have a function that will execute, but hasn't because it hasn't been called to execute. So we have to create an event listener for that to happen.

For you last line of code, you want to add it right after the last curly braces of your function, but still within the <script> as follows:

                
                    window.addEventListener("load", createList, false);

What this is doing is that, when the window loads it wants it to execute the function we created or createList.

Finished Product

Your final should look like this:

                
                    <!doctype html>
                    <html>
                        <body>
                            <ul id="shopping"></ul>
                            <script>
                                var fruit = ["Apples", "Oranges", "Bananas", "Strawberries", "Cherries", "Blueberries", "Watermelons"];
                                function createList() {
                                        for(i = 0; i < fruit.length; i++) {
                                                var list = document.getElementById("shopping");
                                                var item = document.createElement("li");
                                                item.innerHTML = fruit[i];
                                                list.appendChild(item);
                                        }
                                }
                                window.addEventListener("load", createList, false);
                            </script>
                        </body>
                    </html>
